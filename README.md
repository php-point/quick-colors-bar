# Quick Color Palette for VS Code

A VS Code extension that provides a Quick Color Palette for VS Code with copy-to-clipboard functionality.

## Features

- Shows a categorized color palette in the Activity Bar
- Click any color to copy its hex code to the clipboard
- Displays a status bar message when a color is copied
- Supports a large selection of colors across multiple categories

## Usage

1. Open the extension in VS Code (`F5` in the Extension Host).
2. Activate the command palette and run `Show Quick Color Palette for VS Code`.
3. Or use the keyboard shortcut:
   - Windows/Linux: `Ctrl+Shift+P`
   - Mac: `Cmd+Shift+P`
4. Click a color swatch to copy the hex code.

## Extension Details

- Name: `Quick Color Palette for VS Code`
- Command: `color-palette.togglePanel`
- View ID: `colorPalette.view`
